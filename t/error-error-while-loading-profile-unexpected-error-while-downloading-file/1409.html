当你第一次打开MOD管理器，并且找到想要的MOD点击下载时！出现下载SML出错提示！
别担心！我找到了解决方法！

①首先你应该让他弹出此错误一次！

②win+R打开运行 输入这段神秘代码:%LOCALAPPDATA% 如图所示！

点击确定！

③在打开的文件夹中找到 AppData\Local\SatisfactoryModManager\downloadCache\smlVersions\3.1.1文件夹，看看里面有没有东西，有的话就删除清空！

④然后你可以找你的朋友把他的3.1.1文件夹里的 SML.smod 复制给你，粘贴进去就行了！
或者下载我提供的 SML.smod 但是这个网站好像没有办法上传文件并提供下载！以后再说

⑤然后打开Satisfactory游戏文件夹，找到 Satisfactory\FactoryGame\Mods\SML 查看里面是否有文件 和图片一样的哦！

6%25%7D%40%60%5D2%5B%5D16N%604M0YO7EDI81119×626 30.8 KB

如果没有 请到 https://github.com/satisfactorymodding/SatisfactoryModLoader/releases/tag/3.1.1 下载sml.zip 并且解压到MOD文件夹里！

⑥此时再重新打开MOD管理器 然后设置profile:为 development 恭喜你 大功告成！

你可以下载自己喜欢的MOD了 ！

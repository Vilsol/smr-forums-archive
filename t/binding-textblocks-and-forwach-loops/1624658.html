This is what I used to make a list of available recipes. So you should change some things for your use case. I’ll try to note where you should change what.

Create a simple object blueprint, and add (instead of recipe) a text variable. And make sure to check Expose on Spawn

Create a widget, and in its settings add interface User Object List Entry

Adding the interface creates a function Get List Item Object. Create a variable of type object and return it in this function

image821×226 20.2 KB

In the event graph, in event On List Item Object Set (also made available by adding the interface) set the variable you set before to the event param

Create the textbox for the item’s text (here I also have the icon of the recipe, but you don’t have one)

In the function to get the text of the textbox you create, cast the item variable to the object type you created in the first step, then get the text variable of it, and return it.

image1217×384 48.6 KB

In the main widget, add a listview widget and set its Entry Widget Class to the widget you created at step 2

To add items to the list, in your main widget, in construct event, do this (instead of a recipe pin, yours will have a text pin)
